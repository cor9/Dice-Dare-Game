Game Rules

How to Play

  • This is a dice game similar to Deal or No Deal
  • You need a twenty-sided dice (or use an online roller like rolz.org)
  • Two players: a Game Master and a Roller
  • The Roller rolls dice to eliminate dares from the list
  • The Game Master keeps track of rolled numbers
  • When only one dare remains, it must be completed

Roller Advantage

The Roller can end the game early by choosing the "center dare" instead of continuing to roll. This gives them control over when the game ends.

Center Dare Rule

If there's an even number of dares remaining, use the lower numbered dare of the two centermost dares.
